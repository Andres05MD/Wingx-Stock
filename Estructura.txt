Idioma: Español
Tecnologías Principales: Next.js 15+, React 19, TailwindCSS 4, Firebase, SweetAlert2, Lucide React, Date-fns.

Descripción General:
Sistema web para la gestión integral de una tienda de ropa (Wingx). Permite el control de inventario, pedidos, clientes y producción. Incluye un sistema de autenticación multi-usuario y un dashboard administrativo con estadísticas.

Estructura de Directorios (Source):
/src
├── /app                 # Rutas de la aplicación (App Router)
│   ├── /agenda          # Calendario de pedidos
│   ├── /clientes        # Gestión de base de datos de clientes
│   ├── /inventario      # Inventario de productos terminados (Stock)
│   ├── /login           # Pantalla de inicio de sesión
│   ├── /materiales      # Gestión de insumos y materiales
│   ├── /pedidos         # Gestión de pedidos de clientes
│   ├── /prendas         # Catálogo de prendas y costos
│   ├── layout.tsx       # Layout principal (Shell con Sidebar)
│   └── page.tsx         # Dashboard principal
├── /components          # Componentes de interfaz
│   ├── AdminDashboard   # Panel de estadísticas administrativas
│   ├── ClientForm       # Formulario de clientes
│   ├── ExchangeRate...  # Widget de tasa de cambio
│   ├── GarmentForm      # Formulario de prendas
│   ├── OrderForm        # Formulario de pedidos
│   ├── Shell            # Contenedor principal
│   └── Sidebar          # Navegación lateral
├── /context             # Estados globales
│   ├── AuthContext      # Manejo de sesión y usuarios
│   └── ExchangeRate...  # Manejo de tasa de cambio (Bs/$)
├── /lib                 # Configuraciones externas
│   └── firebase.ts      # Inicialización de Firebase
└── /services            # Lógica de negocio
    └── storage.ts       # Servicios de persistencia de datos

Módulos Funcionales:

1. Autenticación (Auth):
   - Inicio de sesión seguro.
   - Identificación de "owner" para separación de datos.
   - Protección de rutas.

2. Dashboard Principal:
   - Estadísticas clave (Usuarios, Pedidos, Productos).
   - Acciones rápidas.
   - Resumen de actividad reciente.

3. Gestión de Prendas:
   - Catálogo de productos que se pueden fabricar.
   - Definición de costos y precios de venta.

4. Gestión de Clientes:
   - Registro de información de contacto.
   - Historial de interacciones.

5. Gestión de Pedidos:
   - Ciclo de vida del pedido (Sin comenzar, Pendiente, En proceso, Entregado).
   - Asociación con clientes y prendas.
   - Visualización visual por estados (Colores).

6. Inventario (Stock):
   - Control de productos listos para entrega inmediata.

7. Materiales:
   - Lista de materiales requeridos para producción.
   - Gestión de compras pendientes.

8. Agenda:
   - Visualización de calendario.
   - Fechas de entrega y organización temporal.

9. Configuración Global:
   - Tasa de cambio global para cálculos de precios (Bs.).
